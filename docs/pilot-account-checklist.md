---

## 계정 생성 방법

### 방법 1: 교사가 직접 회원가입 (권장) ✅

**장점**:
- 프로필 자동 생성 (트리거로 처리)
- 교사가 직접 비밀번호 설정
- 관리자 개입 최소화

**단계**:
1. 교사에게 회원가입 링크 전달
   - URL: https://sunday-school-eta.vercel.app/signup
2. 교사가 직접 가입
   - 이메일, 비밀번호, 이름 입력
   - 자동으로 `teacher` 역할로 프로필 생성
3. 관리자가 담당 반 배정
   - 관리자 페이지 → 반 수정 → 담임 교사 선택

---

### 방법 2: 관리자가 수동 생성

**단계**:
1. Supabase 대시보드 → Authentication → Users
2. "Add user" 클릭
3. 이메일/비밀번호 입력, "Auto Confirm User" 체크
4. Table Editor → profiles에서 프로필 생성
