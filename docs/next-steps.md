### 2단계: 파일럿 준비 (1-2일 내)

**목표**: 실제 사용자를 위한 준비

#### 2.1 파일럿 사용자 계정 생성

- [ ] **교사 계정 생성**
  - Supabase 대시보드 → Authentication → Users
  - "Add user" 클릭
  - 이메일/비밀번호 입력
  - "Auto Confirm User" 체크
  - 프로필 생성 (Table Editor → profiles)
    - `id`: 사용자 UUID
    - `email`: 동일한 이메일
    - `role`: `teacher` 또는 `admin`

- [ ] **계정 정보 전달**
  - 이메일/비밀번호를 안전하게 전달
  - 사용 가이드 링크 제공

**참고 문서**: `docs/setup-admin-user.md`

#### 2.2 실제 데이터 입력 (선택사항)

테스트 데이터 대신 실제 데이터를 사용하려면:

- [ ] **반 데이터 입력**
  - Supabase 대시보드 → Table Editor → classes
  - 실제 반 정보 입력

- [ ] **학생 데이터 입력**
  - Supabase 대시보드 → Table Editor → students
  - 실제 학생 정보 입력

**참고 문서**: `docs/data-input-guide.md`

**예상 소요 시간**: 1-2시간
