# Supabase 프로젝트명 변경 가이드

> Supabase 프로젝트의 이름을 변경하는 방법

---

## 프로젝트명 변경 방법

### 1. Supabase 대시보드 접속

1. [Supabase 대시보드](https://app.supabase.com) 접속
2. 로그인 (필요 시)
3. 프로젝트 선택 (현재 "SudySchool" 또는 다른 이름으로 되어 있는 프로젝트)

### 2. 프로젝트 설정 페이지로 이동

1. 좌측 사이드바에서 **Settings** (⚙️ 아이콘) 클릭
2. **General** 메뉴 클릭

### 3. 프로젝트명 변경

1. **Project Name** 섹션 찾기
2. 현재 프로젝트명 옆의 **편집 버튼** (연필 아이콘 또는 Edit 버튼) 클릭
3. 프로젝트명을 `SundaySchool`로 변경
4. **Save** 또는 **Update** 버튼 클릭

---

## 주의사항

### 변경 사항

- ✅ **프로젝트 이름**: 변경됨 (대시보드에서 표시되는 이름)
- ✅ **프로젝트 URL**: 변경되지 않음 (기존 URL 유지)
- ✅ **API 키**: 변경되지 않음 (기존 키 유지)
- ✅ **데이터베이스**: 변경되지 않음 (기존 데이터 유지)

### 환경 변수 확인

프로젝트명을 변경해도 다음은 변경되지 않습니다:
- `NEXT_PUBLIC_SUPABASE_URL` - 동일 (예: `https://xxxxx.supabase.co`)
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` - 동일
- `SUPABASE_SERVICE_ROLE_KEY` - 동일

따라서 **환경 변수나 코드 수정이 필요하지 않습니다**.

---

## 변경 후 확인

1. 대시보드에서 프로젝트 목록 확인
2. 프로젝트 이름이 "SundaySchool"로 변경되었는지 확인
3. 프로젝트 URL과 API 키가 동일한지 확인 (Settings → API)

---

## 문제 해결

### 프로젝트명 변경 옵션이 보이지 않을 때

- Supabase 무료 플랜에서도 프로젝트명 변경 가능
- Settings → General 페이지에서 확인
- 프로젝트 소유자 권한이 필요 (본인이 만든 프로젝트)

### 변경이 저장되지 않을 때

- 페이지 새로고침 후 다시 시도
- 브라우저 캐시 삭제 후 시도
- Supabase 지원팀에 문의

---

## 참고

- 프로젝트명은 대시보드에서만 표시되는 이름입니다
- 실제 프로젝트 URL (예: `https://xxxxx.supabase.co`)은 변경되지 않습니다
- 환경 변수나 코드 변경이 필요하지 않습니다
