# GitHub & Vercel 무료 플랜 제한 사항

> 파일럿 및 초기 운영 시 알아두면 좋은 무료 플랜 제한 사항

---

## GitHub 무료 플랜

### 기본 제공 사항 (2024 기준)

- ✅ **Private 저장소**: 무제한
- ✅ **Public 저장소**: 무제한
- ✅ **협업자**: 무제한 (Private 저장소 포함)
- ✅ **저장 용량**: 500MB (저장소당)
- ✅ **대역폭**: 
  - Private 저장소: 1GB/월
  - Public 저장소: 무제한
- ✅ **Actions (CI/CD)**: 
  - Private 저장소: 2,000분/월
  - Public 저장소: 무제한

### 이 프로젝트에 적용

- ✅ **충분함**: Private 저장소 무제한, 협업자 무제한
- ✅ **코드 용량**: Next.js 프로젝트는 보통 50-200MB 이내
- ✅ **대역폭**: 소규모 사용 시 충분 (1GB/월)
- ⚠️ **주의**: 큰 파일(이미지, PDF 등)은 Git LFS 사용 권장

## Vercel 무료 플랜 (Hobby)

### 기본 제공 사항 (2024 기준)

- ✅ **배포**: 무제한
- ✅ **대역폭**: 100GB/월
- ✅ **빌드 시간**: 6,000분/월
- ✅ **서버리스 함수 실행 시간**: 100GB-시간/월
- ✅ **도메인**: Vercel 도메인 무료 (예: `your-project.vercel.app`)
- ✅ **SSL 인증서**: 자동 (무료)
- ✅ **CDN**: 전 세계 CDN 무료
- ✅ **커스텀 도메인**: 무제한 (도메인은 별도 구매 필요)

### 이 프로젝트에 적용

- ✅ **충분함**: 
  - 파일럿 단계: 사용자 수가 적어 대역폭 충분
  - 빌드 시간: Next.js 빌드는 보통 1-3분 소요
  - 월 2,000회 이상 빌드 가능 (실용적으로 충분)
- ✅ **서버리스 함수**: Supabase를 사용하므로 서버리스 함수 사용 최소
- ⚠️ **주의**: 
  - 대역폭 100GB/월 초과 시 유료 플랜 필요 (월 $20부터)
  - 월 활성 사용자가 수천 명 이상일 때 고려

## Supabase 무료 플랜

### 기본 제공 사항 (2024 기준)

- ✅ **데이터베이스**: 500MB 저장 용량
- ✅ **대역폭**: 5GB/월 (출력)
- ✅ **API 요청**: 50,000건/월
- ✅ **인증 사용자**: 50,000명/월
- ✅ **파일 저장**: 1GB
- ✅ **실시간 연결**: 200개 동시 연결

### 이 프로젝트에 적용

- ✅ **파일럿 단계**: 충분
  - 학생 수: 수십~수백 명
  - 출석 체크: 일주일 1회 = 월 4회
  - API 요청: 월 수천 건 이내
- ⚠️ **주의**:
  - 데이터베이스 용량 500MB 초과 시 유료 플랜 필요
  - 학생 프로필 이미지 등 큰 파일은 Supabase Storage 사용 권장

## 예상 사용량 (파일럿 단계)

### GitHub
- **저장소 용량**: ~100MB (코드 + 문서)
- **대역폭**: 월 수십 MB (배포 시)
- **결론**: ✅ 무료 플랜으로 충분

### Vercel
- **대역폭**: 
  - 사용자 10명 × 일 10회 접속 × 월 30일 = 3,000회
  - 페이지당 평균 1MB = 3GB/월
- **빌드 시간**: 
  - 배포 시 1-2분
  - 월 10회 배포 = 10-20분
- **결론**: ✅ 무료 플랜으로 충분

### Supabase
- **데이터베이스**: 
  - 학생 100명 × 평균 5KB = 500KB
  - 출석 기록 1년치 = 수 MB
- **API 요청**: 
  - 사용자 10명 × 일 50회 = 500회/일 = 15,000회/월
- **결론**: ✅ 무료 플랜으로 충분

## 유료 플랜이 필요한 시점

### GitHub
- ❌ 파일럿/초기 단계에서는 불필요
- ✅ 필요 시: 
  - 큰 파일 (비디오, 대용량 이미지) 저장
  - 높은 Actions 사용량 (복잡한 CI/CD)

### Vercel
- ❌ 파일럿/초기 단계에서는 불필요
- ✅ 필요 시:
  - 월 활성 사용자 1,000명 이상
  - 대역폭 100GB/월 초과
  - 팀 협업 기능 필요

### Supabase
- ❌ 파일럿/초기 단계에서는 불필요
- ✅ 필요 시:
  - 학생 수 1,000명 이상
  - 데이터베이스 500MB 초과
  - 월 API 요청 50,000건 초과
  - 파일 저장 1GB 초과
