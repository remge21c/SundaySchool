## 시작하기

### 1. 로그인

1. 웹 브라우저에서 시스템 URL 접속 (관리자가 안내한 주소)
2. **로그인** 페이지에서 계정 정보 입력:
   - **이메일**: 관리자가 발급한 이메일 주소
   - **비밀번호**: 관리자가 발급한 임시 비밀번호
3. **로그인** 버튼 클릭

### 2. 첫 사용 시 온보딩

첫 로그인 시 **온보딩 튜토리얼**이 자동으로 시작됩니다.

튜토리얼에서는 다음을 안내합니다:
- 반 선택 방법
- 출석 체크 방법
- 심방 기록 방법

약 3분 소요되며, 필요 시 **건너뛰기**도 가능합니다.

### 3. 대시보드

로그인 후 **대시보드** 페이지로 자동 이동합니다.

대시보드에서 확인할 수 있는 내용:
- **빠른 액션 카드**: 출석 체크, 학생 관리
- **장기 결석 알림**: 3주 이상 결석한 학생 목록

## 출석 체크하기

출석 체크는 **모바일에서도 쉽게** 할 수 있습니다.

### 1. 출석 체크 페이지 접속

- 대시보드에서 **출석 체크** 카드 클릭
- 또는 사이드바에서 **출석 체크** 메뉴 클릭

### 2. 반 선택

화면 왼쪽(모바일에서는 상단 버튼)에서 담당 반을 선택합니다.

- 부서별로 그룹화되어 표시됩니다 (예: 유년부, 초등부)
- 부서를 클릭하면 하위 반 목록이 표시됩니다
- 반을 클릭하면 오른쪽에 학생 목록이 표시됩니다

### 3. 날짜 선택

오른쪽 상단의 **날짜 선택** 카드에서 출석을 체크할 날짜를 선택합니다.

- 기본값은 오늘 날짜입니다
- 과거 날짜도 선택 가능합니다

### 4. 출석 상태 변경

각 학생 카드를 **터치(클릭)**하면 출석 상태가 순환됩니다:

1. **미체크** (회색) → 클릭
2. **출석** (초록색) → 클릭
3. **지각** (노란색) → 클릭
4. **결석** (빨간색) → 클릭
5. **출석**으로 다시 돌아감

### 5. 결석 사유 입력

**결석**을 선택하면 사유 입력 창이 자동으로 나타납니다.

다음 중 하나를 선택하거나 직접 입력할 수 있습니다:
- 아픔
- 여행
- 늦잠
- 기타 (직접 입력)

**확인** 버튼을 클릭하면 저장됩니다.

### 6. 출석 통계 확인

학생 목록 상단에 **출석 통계**가 표시됩니다:
- 전체 학생 수
- 출석 수
- 결석 수
- 지각 수
- 출석률

## 학생 정보 확인하기

### 1. 학생 프로필 접속

**방법 1: 출석 체크 페이지에서**
- 출석 체크 페이지에서 학생 카드를 클릭

**방법 2: 장기 결석 알림에서**
- 대시보드의 장기 결석 알림 카드에서 학생을 클릭

### 2. 학생 정보 확인

학생 프로필 페이지에서 다음 정보를 확인할 수 있습니다:

- **기본 정보**
  - 이름, 학년, 생년월일, 성별
  - 보호자 연락처
  - 주소, 학교명
  
- **알레르기 정보** (있는 경우)
  - 음식 알레르기
  - 약물 알레르기
  - 기타

### 3. 심방 기록 확인

학생 프로필 하단에 **심방 타임라인**이 표시됩니다.

- 날짜순으로 최신 심방 기록이 위에 표시됩니다
- 심방 유형 아이콘 (전화, 심방, 카카오톡)
- 비밀 보장 표시 (자물쇠 아이콘)
- 기도 제목 (있는 경우)

---

## 심방 기록하기

### 1. 학생 프로필 접속

출석 체크 페이지나 장기 결석 알림에서 학생을 클릭하여 프로필 페이지로 이동합니다.

### 2. 심방 기록 폼 사용

학생 프로필 하단의 **심방 기록 폼**에서 다음 정보를 입력합니다:

- **날짜**: 심방한 날짜 선택
- **유형**: 전화 / 심방 / 카카오톡 중 선택
- **내용**: 심방 내용 입력 (필수)
- **기도 제목**: 기도 제목 입력 (선택)
- **비밀 보장**: 민감한 내용인 경우 체크

### 3. 저장

**저장** 버튼을 클릭하면 심방 기록이 저장되고 타임라인에 표시됩니다.

## 장기 결석 알림 확인하기

### 대시보드에서 확인

대시보드의 **장기 결석 알림** 카드에서 3주 이상 결석한 학생 목록을 확인할 수 있습니다.

- 학생 이름과 마지막 출석 날짜가 표시됩니다
- 학생을 클릭하면 해당 학생의 프로필 페이지로 이동합니다

### 사이드바에서 확인

사이드바의 **장기 결석 알림** 메뉴 옆에 빨간 배지로 결석 학생 수가 표시됩니다.

## 자주 묻는 질문

### Q1: 출석을 잘못 입력했어요. 어떻게 수정하나요?

A: 출석 체크 페이지에서 학생 카드를 다시 클릭하여 올바른 상태로 변경하면 됩니다. 마지막 입력한 상태가 저장됩니다.

### Q2: 과거 날짜의 출석을 체크할 수 있나요?

A: 네, 출석 체크 페이지에서 날짜를 변경하여 과거 날짜의 출석도 체크할 수 있습니다.

### Q3: 다른 반의 학생 정보를 볼 수 있나요?

A: 담당하지 않은 반의 학생 정보는 볼 수 없습니다. 자신이 담당하는 반의 학생 정보만 확인할 수 있습니다.

### Q4: 심방 기록을 삭제할 수 있나요?

A: 현재 버전에서는 심방 기록 삭제 기능이 없습니다. 오류가 있는 경우 관리자에게 문의하세요.

### Q5: 비밀번호를 변경하고 싶어요.

A: 현재 버전에서는 비밀번호 변경 기능이 없습니다. 비밀번호 변경이 필요하면 관리자에게 문의하세요.

### Q6: 모바일에서 사용할 수 있나요?

A: 네, 스마트폰 브라우저에서도 사용할 수 있습니다. 모바일에 최적화되어 있습니다.

### Q7: 오프라인에서 사용할 수 있나요?

A: 현재 버전에서는 인터넷 연결이 필요합니다. 오프라인 기능은 향후 추가될 예정입니다.

## 문제 발생 시

### 로그인 문제

- 이메일과 비밀번호를 다시 확인하세요
- 문제가 계속되면 관리자에게 문의하세요

### 데이터가 보이지 않을 때

- 반을 올바르게 선택했는지 확인하세요
- 새로고침(F5)을 시도해보세요
- 문제가 계속되면 관리자에게 문의하세요

### 오류 메시지가 나타날 때

- 에러 메시지 내용을 메모해두세요
- 새로고침(F5)을 시도해보세요
- 문제가 계속되면 에러 메시지와 함께 관리자에게 문의하세요

## 팁

### 빠른 출석 체크

1. 출석 체크 페이지 접속
2. 반 선택 (자주 사용하는 반은 기억해두세요)
3. 학생 카드를 빠르게 터치하여 출석 상태 변경
4. 목표: 3분 이내 완료

### 심방 기록 습관화

- 심방 후 즉시 기록하는 것을 권장합니다
- 비밀 보장 체크박스를 적절히 활용하세요
- 기도 제목도 함께 기록하면 더 좋습니다

### 장기 결석 알림 활용

- 주기적으로 대시보드의 장기 결석 알림을 확인하세요
- 3주 이상 결석한 학생에게 관심을 기울이세요
- 심방 기록을 통해 지속적으로 돌봄을 제공하세요
